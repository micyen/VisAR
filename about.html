About

VisAR, short for Visualizer for Adverse Reactions, is a project for 16:198:526 - Data Interaction and Visual Analytics at Rutgers University, taught by Professor James Abello.

This project is authored by Michael Yen (front end / visualization / interactivity) and Peter Krasucki (back end / minimization algorithms / data wrangling).

The goal of this project is to facilitate visual analytics of the FDA Adverse Event Reporting System (FAERS) dataset, publicly available at https://fis.fda.gov/extensions/FPD-QDE-FAERS/FPD-QDE-FAERS.html, using interactive graphs and charts. Hover over the help icon on any page to see a description for that visualization.

Libraries used to build this project include d3.js, 3d-force-graph, plotly.js, and noUiSlider. Programming languages include Python, HTML/CSS and JavaScript.
